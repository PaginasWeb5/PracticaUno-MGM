Para comenzar veremos algunos conceptos básicos que hay que conocer sobre HTML, y estas siglas significan HypeText Makeup Lenguage, osea lenguaje marcado de texto.

Tablas

Para la crear una tabla necesitaremos utilizar la etiqueta "table" y en este se encuentran otras etiquetas como el tb, th y tr.

La etiqueta "th" funciona de la misma forma que la etiqueta "td" pero esta etiqueta tiene la diferencia que su contenido está considerado como el encabezado de la tabla.

"tr" esta etiqueta se utiliza para las filas de una tabla.

"td", la etiqueta td se usa para las celdas de la tabla.

*CSS*

Las siglas de CSS significan Cascading Style Sheets, Hojas de Estilo de Cascadas. Es un lenguaje que describe el estilo de un documento HTML.
Existen diferentes tipos de estilos en CSS y estos son:

  • Inline
  • Interno
  • Externo

Inline: Unifica en una declaración todos los estilos para un archivo html

Interno: Esta incorporada a un documento HTML a través del elemento style

Externo: Se almacenan en un archivo diferente al del archivo con el código HTML, el cual estan vinculado mediante el elemento link


En CSS existen los selectores que su funcionamiento es seleccionar todos los elementos que contendrá la página. Algunos de los selectores son:

  • Etiqueta: Este selector esta también común porque su función es seleccionar a todos los elementos que contengan la etiqueta HTML. Sintaxis: Etiqueta html{ propiedad: valor;}
  • Clase: Sirve para dar estilo sólo a determinados elementos marcados con el atributo class y la clase definida. Sintaxis:.nombreDeLaClase{ propiedad: valor;}
  • ID: Iguales que los anteriores pero para elementos marcados con el atributo id. Este se utiliza para marcar un elemento único. Sintaxis: #nombre-del-ID{ propiedad: valor;}
  • Descendente: Selecciona los elementos que se encuentran dentro de otros elementos. Sintaxis:selector1 selector2}{ propiedad: valor; }

Pseudo Clases:

  • Link se aplica a todos los enlaces que todavía no han sido visitados por el usuario.
  • Visited se aplica a todos los enlaces que han sido visitados al menos una vez por el usuario.

A continuación se indican las acciones del usuario que activan cada pseudo-clase:

  • :hover: Se activa cuando el usuario pasa el ratón o cualquier otro elemento apuntador por encima de un elemento.
  • :active: Se activa cuando el usuario activa un elemento, por ejemplo cuando pulsa con el ratón sobre un elemento.
  • :focus: Se activa cuando el elemento tiene el foco del navegador, es decir, cuando el elemento está seleccionado.

Presudo Elementos

Los pseudo elementos se añaden a los selectores, pero, a diferencia de las pseudo clases, los pseudo elementos no describen un estado sino que permiten añadir estilos a una parte concreta del documento. Algunos presudo elementos son:

  • :first-line: Permite seleccionar la primera línea de texto de un elemento.
  • :first-letter: Permite seleccionar la primera letra de la primera línea de texto de un elemento.
  • :before y :afterSe utilizan en combinación con la propiedad content de CSS para añadir contenidos antes o después del contenido original de un elemento.
